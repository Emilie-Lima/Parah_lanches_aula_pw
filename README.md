# Pará Lanches

O Pará Lanches é um projeto simples desenvolvido com o objetivo de praticar conceitos básicos de desenvolvimento web utilizando HTML, CSS e JavaScript. A proposta é simular um sistema de login que direciona o usuário para uma página de exibição de produtos, funcionando como uma base inicial para aplicações maiores.

O sistema conta com uma tela de login com campos de usuário e senha, cuja verificação é feita por meio de uma função em JavaScript chamada `verificarLogin()`. Essa validação é apenas simulada no lado do cliente, sem qualquer integração com backend. Após o login, o usuário pode acessar uma página onde é exibido um produto, com um efeito visual de zoom aplicado ao passar o mouse sobre a imagem, trazendo um pouco mais de interatividade para a interface.

A estrutura do projeto é organizada em arquivos separados, incluindo páginas HTML (`index.html`, `teste.html` e `produtos.html`), um arquivo de estilos (`styles.css`) responsável pela aparência da aplicação, e um script JavaScript que controla a lógica de login. Além disso, há uma pasta destinada às imagens utilizadas no site.

O layout foi desenvolvido de forma simples, com elementos centralizados e uso de cores mais chamativas, buscando facilitar a visualização e a usabilidade. Apesar de funcional, o projeto ainda pode ser expandido com diversas melhorias, como a implementação de um sistema de autenticação real com backend, adição de mais produtos, criação de um carrinho de compras, melhorias na responsividade para dispositivos móveis e aprimoramento geral da interface.

Para executar o projeto, basta abrir o arquivo `index.html` em um navegador. Este trabalho foi desenvolvido como prática de aprendizado em desenvolvimento web.
